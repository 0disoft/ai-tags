# Change Log

All notable changes to the "ai-tags" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [1.1.0] - 2026-01-21 KST

### Added

- @AI:SYNC now supports line navigation with `:L123` or `:L10-L20` syntax.
- @AI:SYNC now supports symbol navigation with `#functionName` or `#ClassName.methodName` syntax.
- Hover links and Inlay hints now navigate to the specified line or symbol.

## [1.0.1] - 2026-01-19 UTC

- Added AI tag diagnostics for EXPIRY and SYNC.
- Added SYNC hover links with create shortcuts for missing targets.
- Added tag keyword highlighting.
- Added AI Tags explorer view for workspace tag navigation.
